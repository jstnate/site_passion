Le Coding

Magie ou pure logique

Qu'est-ce que c'est ?

le Coding est la pratique par laquelle on écrit un programme informatique

La programmation, appelée aussi "code" dans le domaine informatique, désigne l'ensemble des activités qui permettent l'écriture des programmes informatiques. C'est une étape importante du développement de logiciels, de sites web, d'objets connectés. L'écriture d'un programme se fait dans un langage de programmation. Le code est présent partout autour de nous que ce soit dans les télés, les téléphones ou encore les voitures
